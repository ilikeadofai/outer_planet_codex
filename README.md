# 외계 행성 생태계 에너지 시뮬레이터

가상의 외계 행성 환경을 설정하고, 먹이 단계별 에너지 흐름을 한눈에 이해할 수 있는 고등학교 과학 발표용 웹앱입니다.

## 주요 기능
1. **환경 설정 시뮬레이션**
   - 황화수소(H2S), 온도, 압력, 빛의 양을 슬라이더로 조정할 수 있습니다.
2. **생물 선택 기반 생태계 구성**
   - 생산자 / 1차 소비자 / 2차 소비자를 카드로 선택해 조합을 바꿔볼 수 있습니다.
3. **에너지 피라미드 시각화**
   - 단계별 에너지 양과 손실량을 막대 UI로 직관적으로 확인할 수 있습니다.
4. **발표용 설명 자동 생성(선택 기능)**
   - API 키를 입력하면 현재 설정을 바탕으로 한국어 발표 설명문을 생성할 수 있습니다.

## 기술 스택
- **React + Vite**
- **Tailwind CSS**
- **GitHub Actions + GitHub Pages**

## 실행 방법
```bash
npm install
npm run dev
```

## GitHub Pages 배포 방법
이 프로젝트는 GitHub Actions로 자동 배포됩니다.

1. 저장소 기본 브랜치를 `main`으로 사용합니다.
2. GitHub 저장소에서 **Settings → Pages**로 이동합니다.
3. Source를 **GitHub Actions**로 선택합니다.
4. `main` 브랜치에 push 하면 자동으로 배포됩니다.

배포 링크 형식:
```text
https://<github-username>.github.io/<repository-name>/
```

## 보안 주의
- OpenAI/OpenRouter API 키는 **브라우저 세션(React state)에서만 사용**됩니다.
- localStorage, sessionStorage, 쿠키에 저장하지 않으므로 새로고침하면 키가 사라집니다.

## 발표에서 강조하면 좋은 포인트
1. **에너지 전달의 핵심 원리**: 먹이 단계가 올라갈수록 사용 가능한 에너지가 크게 줄어든다는 점
2. **환경-생태계 연결성**: 같은 생물 조합도 환경 조건에 따라 결과가 달라진다는 점
3. **교육용 시각화의 장점**: 복잡한 식보다 그래프/요약 문장으로 과학 개념을 쉽게 전달할 수 있다는 점
